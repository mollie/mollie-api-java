# SalesInvoiceVatScheme

The VAT scheme to create the invoice for. You must be enrolled with One Stop Shop enabled to use it.

## Example Usage

```java
import com.mollie.mollie.models.components.SalesInvoiceVatScheme;

SalesInvoiceVatScheme value = SalesInvoiceVatScheme.STANDARD;
```


## Values

| Name            | Value           |
| --------------- | --------------- |
| `STANDARD`      | standard        |
| `ONE_STOP_SHOP` | one-stop-shop   |
