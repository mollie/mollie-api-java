# SalesInvoicePaymentDetailsSource

The way through which the invoice is to be set to paid.

## Example Usage

```java
import com.mollie.mollie.models.components.SalesInvoicePaymentDetailsSource;

SalesInvoicePaymentDetailsSource value = SalesInvoicePaymentDetailsSource.MANUAL;
```
