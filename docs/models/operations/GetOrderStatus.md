# GetOrderStatus

The order's status. Refer to the [documentation regarding statuses](order-status-changes) for more info about
which statuses occur at what point.


## Values

| Name         | Value        |
| ------------ | ------------ |
| `CREATED`    | created      |
| `PENDING`    | pending      |
| `AUTHORIZED` | authorized   |
| `PAID`       | paid         |
| `SHIPPING`   | shipping     |
| `CANCELED`   | canceled     |
| `EXPIRED`    | expired      |
| `COMPLETED`  | completed    |
