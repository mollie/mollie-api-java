# GetOrderCategory

The voucher category, in case of an order line eligible for a voucher. See the
[Integrating Vouchers](integrating-vouchers) guide for more information.


## Values

| Name            | Value           |
| --------------- | --------------- |
| `MEAL`          | meal            |
| `ECO`           | eco             |
| `GIFT`          | gift            |
| `SPORT_CULTURE` | sport_culture   |
